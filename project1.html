IoT Weather Station

By Ankan Bhowmik

Concept

The IoT Weather Station collects real-time environmental data using temperature, humidity, and pressure sensors. It transmits data to a cloud server for remote monitoring and analysis.

Components Required

  • DHT11 Temperature & Humidity Sensor
  • ESP8266 Wi-Fi Module
  • Arduino Uno or NodeMCU
  • Resistors (1kŒ©, 10kŒ©)
  • LEDs (Red, Green)
  • Jumper Wires & Breadboard

Connections

  • Connect the anode (long leg) of the Red LED to digital pin 9 on the Arduino with a 1kŒ© resistor.
  • Connect the other end of the resistor to the ground (GND) pin on the Arduino.
  • Connect the anode (long leg) of the Green LED to digital pin 8 on the Arduino with a 1kŒ© resistor.
  • Connect the other end of the resistor to the ground (GND) pin on the Arduino.
  • Connect the GND pin on the Arduino to the negative rail on the breadboard.

Arduino Code


// Sample Arduino code for IoT Weather Station

void setup() {
    pinMode(9, OUTPUT);
    pinMode(8, OUTPUT);
}

void loop() {
    digitalWrite(9, HIGH);
    digitalWrite(8, LOW);
    delay(30000); // Wait for 30 seconds

    digitalWrite(8, HIGH);
    digitalWrite(9, LOW);
    delay(30000); // Wait for 30 seconds
}
