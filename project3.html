LED Controller by Arduino Uno R3

By Arghya Roy

üìÖ Post Date: January 23, 2025

Concept

By using the microcontroller how we can operate the blinking of LEDs in a span of time.

Components Required

  • Arduino Uno R3
  • LED
  • Resistors
  • Breadboard
  • Jumper wires
  • Power supply

Connections

  • Connect the Arduino GND pin to the breadboard ground rail.
  • Connect the Arduino 5V pin to the breadboard power rail.
  • Place the red, yellow, and green LEDs on the breadboard, with one leg of each connected to the ground rail.
  • Connect the other leg of the red LED to digital pin 6.
  • Connect the other leg of the yellow LED to digital pin 3.
  • Connect the other leg of the green LED to digital pin 0.
  • Ensure all ground legs of the LEDs are connected to the ground rail, and confirm the ground rail is linked to the Arduino GND pin.

Arduino Code


// C++ code

int animationspeed = 0;

void setup() {
    pinMode(LED_BUILTIN, OUTPUT);
    pinMode(6, OUTPUT);
    pinMode(3, OUTPUT);
    pinMode(0, OUTPUT);
}

void blink(int pin, int speed) 
    digitalWrite(pin, HIGH);
    delay(speed);
    digitalWrite(pin, LOW);
    delay(speed);
}

void loop() {
    animationspeed = 400;
    blink(LED_BUILTIN, animationspeed);
    blink(6, animationspeed);
    blink(3, animationspeed);
    blink(0, animationspeed);
}
