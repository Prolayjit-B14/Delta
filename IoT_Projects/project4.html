Automatic Light Control using LDR

By Ankan Bhowmik

üìÖ Post Date: January 23, 2025

Concept

An automatic light control system can be implemented using an LDR (light-dependent resistor) and Arduino Uno R3. The LDR senses ambient light levels and sends the data to the Arduino, which then controls a connected light. When the ambient light level drops below a certain threshold, the Arduino turns the light on, and when the light level rises above the threshold, it turns the light off. Practical Example: This system can be used for streetlights that automatically turn on at dusk and off at dawn, ensuring energy efficiency and reducing the need for manual control.

Connections

  • Connect the Arduino GND pin to the breadboard ground rail.
  • Connect the Arduino 5V pin to the breadboard power rail.
  • Place the LDR on the breadboard and connect one leg to the 5V rail.
  • Connect a 1 k‚Ñ¶ resistor from the other leg of the LDR to the ground rail.
  • Connect the junction of the LDR and the resistor to an analog input pin (A0) on the Arduino.
  • Connect the positive leg (anode) of the LED to a digital pin (e.g., 6) on the Arduino.
  • Connect the negative leg (cathode) of the LED to the ground rail on the breadboard.

Arduino Code


// C++ code

#define relay 10
int LED = 9;
int LDR = A0;

void setup()
{
pinMode(LED, OUTPUT);
pinMode(relay, OUTPUT);
pinMode(LDR, INPUT);
}

void loop()
{
int LDRValue = analogRead(LDR);
if (LDRValue <=200) //Adjust the range for LDRValue
{
digitalWrite(LED, HIGH);
digitalWrite(relay, HIGH);
}
else
{
digitalWrite(LED, LOW);
digitalWrite(relay, LOW);
}
}
