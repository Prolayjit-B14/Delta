Smart Home Automation

By Ankan Bhowmik

üìÖ Post Date: January 19 , 2025

üå± Key Concept

This Smart Home Automation System, developed using Arduino on Tinkercad, automates lighting and fan control based on motion and temperature. The system uses a PIR sensor to detect motion and a TMP36 temperature sensor to monitor ambient temperature, displaying real-time status on an LCD screen. With components like the Arduino Uno R3, SPDT relay module, and a 12V DC motor, it offers a seamless user experience. This project could be further enhanced with IoT integration for remote control and expanded functionality.
